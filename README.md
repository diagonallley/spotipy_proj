A Python-based console application to control your Spotify playback with a stylish CLI interface. This project uses `spotipy` to interact with Spotifyâ€™s API and `rich` to enhance the user experience with colored text and interactive prompts.

## Features

- Display the current playing track with title and artist.
- Show playback progress and playback state.
- Interactive playback controls:
  - **Next Track**: Skip to the next song.
  - **Previous Track**: Go back to the previous song.
  - **Play/Pause**: Start or pause playback.
  - **Shuffle**: Toggle shuffle mode.
  - **Quit**: Exit the application.
- Enhanced console output using `rich` for styled text and organized prompts.

## Requirements

- **Python 3.x**
- **Python Libraries**:
  - `spotipy` for Spotify API integration.
  - `rich` for styled console output.
